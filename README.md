# Презентація: Правила, команди та навички для Agentic IDE

Презентація створена за допомогою [Slidev](https://sli.dev/).

## Запуск

```bash
# Встановити залежності
npm install

# Запустити dev сервер
npm run dev

# Зібрати статичну версію
npm run build

# Експортувати PDF
npm run export
```

## Структура

- `slides.md` - основна презентація
- `package.json` - залежності проєкту

## Навігація

- `←` / `→` - наступний/попередній слайд
- `f` - повноекранний режим
- `o` - огляд всіх слайдів
- `g` - перехід до слайду за номером
